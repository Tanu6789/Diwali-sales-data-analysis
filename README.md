# 🪔 Diwali Sales Data Analysis

## 📊 Project Overview
This project is an Exploratory Data Analysis (EDA) on Diwali sales data using Python. The goal is to understand customer purchasing behavior, identify key trends, and generate actionable business insights to improve festive season sales strategies.

---

## 🛠️ Tools & Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Jupyter Notebook

---

## 📁 Project Workflow
- Data loading and inspection
- Data cleaning and handling missing values
- Exploratory Data Analysis (EDA)
- Customer demographic analysis
- State-wise and product-wise sales analysis
- Data visualization using graphs and charts

---

## 📈 Key Insights
- Analyzed customer purchasing patterns based on age, gender, and occupation
- Identified top-performing states and regions in sales
- Found most popular product categories during Diwali season
- Observed spending behavior across different customer segments
- Generated visual insights to support business decision-making

---

## 📊 Visualizations
This project includes various visualizations such as:
- Sales distribution by gender
- State-wise sales analysis
- Product category performance
